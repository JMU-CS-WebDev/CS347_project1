1. I chose to create a website for my sorority, Alpha Omega Epsilons(AOE), chapter here at JMU. I hope that it can continue to be improved and used for my chapter in the future.
  2. I created all the pages on my website and did not get them from anywhere else. The images that I used were either taken by an active member of my sorority and I was given permission to use them or I downloaded from the official Alpha Omega Epsilon sorority site. As an active member I am allowed to use their images.
  3. My site has five pages. One of the files is named index.html and this is the home page of my website. I also have a file named active-members.html, current-exec.html, history.html, and values-facts.html. Each of these serves a specific purpose and is named for that purpose.
  4. At the top of each of my pages there is a consistent header that includes the AOE logo. There is also a navigation bar in this header.
  5. There is a navigation bar that is consistent among all the pages. It is part of the header. In it, there are links to each of the pages on the website that can be found either as the Home button or in a labeled dropdown menu.
  6. My site changed layouts depending on the screen size using media queries. One example is In current-exec.html and active-members.html where the images that had been a row turn into a column.

  9. Every page has an AOE favicon that changes depending on what software you are accessing the website on.
  10. All of the images that I used are placed in an images folder. All of my css files that I used are in a folder labeled css. I did not use any javaScript but there is an empty folder names js
